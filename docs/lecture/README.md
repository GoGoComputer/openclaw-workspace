# 📚 고양이 자경단 · 8시간 자동화 강의 대본

> openclaw-workspace 저장소를 8시간 안에 비개발자도 따라올 수 있게 가르치는 강사용 발화 대본 묶음입니다.
> 개발자 수준의 깊이는 각 시간 끝의 **👨‍💻 개발자 노트** 박스에서 보충합니다.

---

## 이 폴더는 뭔가요

이 폴더의 H1~H8 파일은, 강사가 마이크 앞에 앉아서 **그대로 읽을 수 있는 한 사람의 말 그대로의 대본**입니다.
한 시간씩 한 덩어리, 끊지 않고 이야기하듯 이어집니다. 슬라이드 없이도 진행 가능합니다.

- **대상**: 비개발자(터미널 처음) ~ 중급 개발자 동시 수용
- **본문**: 비개발자 눈높이의 한국어 발화 (비유와 일화 중심)
- **개발자 노트**: 각 파일 마지막 박스에서 내부 동작·디자인 결정 보충
- **호흡**: 시간당 약 19,000~21,000자(공백 제외) 분량의 순수 발화

## 강사 사전 준비

강의 **하루 전**까지 다음을 준비해 주세요.

- 시연용 맥 1대 (수강생과 동일 환경, `./openclaw doctor` 모두 초록불)
- 가능하면 깨끗한 맥 1대 추가 (3교시 처음부터 설치 시연용)
- 외부 모니터 + 어댑터, 마이크
- 터미널 폰트 16pt 이상 (Cmd + + 로 키워두기)
- 6교시용 샘플 백업 파일 1개 미리 만들어 두기 (`./openclaw backup`)
- 5교시용 `surf` 명령 1회 사전 실행 (인터넷 끊겼을 때 캐시 보여주기)
- [README.md](../../README.md), [docs/QUICKSTART-ko.md](../QUICKSTART-ko.md), [docs/ARCHITECTURE.md](../ARCHITECTURE.md) 탭에 미리 열어두기

## 수강생 사전 준비

강의 **하루 전**까지 수강생에게 안내해 주세요.

- macOS 15(세쿼이아) 이상이 도는 맥북 또는 맥미니
- 가능하면 Apple Silicon(M1/M2/M3/M4), 메모리 16GB 이상, 디스크 여유 20GB 이상
- Apple ID 로그인 완료, 관리자 비밀번호 알고 있기
- 안정적인 와이파이, 충전기 지참
- 마음가짐: 명령어 외울 필요 없음, 틀려도 됨, 모르면 즉시 손 들기

수강생이 강의 전에 미리 한 번 쳐볼 수 있는 사전 검증 명령:

```bash
sw_vers
git --version
curl --version | head -1
```

문제가 나도 괜찮습니다. 3교시에서 같이 해결합니다.

## 라이브 진행 팁

- **대본은 그대로 읽어도 됩니다.** 직접 말하는 게 자연스러운 분은 말투만 자기화하셔도 좋습니다.
- **각 교시 마지막 박스(👨‍💻 개발자 노트)** 는 비개발자 비중이 높은 회차에서는 그냥 "심화 관심 있는 분만 따로 봐주세요" 하고 건너뛰셔도 됩니다.
- **수강생이 막히면 절대 다음으로 넘어가지 마세요.** 한 분이 손 들면, 그 한 분이 따라오실 때까지 같이 갑니다. 1교시에서 한 약속입니다.
- **5교시(서프)와 8교시(적용)는 절대 자르지 마세요.** 이 두 시간이 있어야 강의가 의미가 있습니다. 시간이 부족하면 4교시 명령 카탈로그를 압축하세요.
- **빨간 에러가 떠도 당황하지 마세요.** "다시 한 번 돌려보세요" 가 1차 답변, `./openclaw doctor` 가 2차 답변, `./openclaw logs <서비스>` 가 3차 답변입니다. 7교시에서 다루는 멱등성 덕분에 다시 돌리는 게 안전합니다.

---

## 강의가 끝난 뒤

수강생에게 다음 세 가지를 약속받아 주세요.

1. **첫 일주일 동안 매일 한 번** 명령어 한 줄이라도 치기 (`./openclaw doctor` 도 OK)
2. **한 달 안에 자기 일 한 가지**를 자동화로 옮기기 (작은 거 하나면 충분)
3. **한 사람 이상에게** 자기 자동화 사례를 자랑하기 (전파)

이 세 가지가 지켜지면 8시간 강의가 진짜로 살아남습니다.
