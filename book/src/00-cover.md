## 이 책의 약속

1. **이 책 한 권으로 재현 가능합니다.** 인터넷에서 레포를 받지 못하는 환경에서도, 부록 A의 스크립트 전문을 그대로 옮겨 적으면 같은 결과를 얻을 수 있습니다.
2. **코드는 절대 생략하지 않습니다.** `# ... 생략 ...` / `# (이하 동일)` / `...` 같은 표기는 이 책 어디에도 없습니다.
3. **비개발자도 끝까지 갑니다.** 강의 대본의 따뜻한 말투를 그대로 살렸습니다.

## 이 책이 다루는 것

- macOS 기반 자동화 환경(OpenClaw)을 처음부터 설치하고 운용하는 법
- 도커·올라마·웹 자동화(Surf)·로컬 LLM 모델 관리
- 설치 한 줄(`./openclaw install`) 안에서 일어나는 여덟 단계의 진짜 의미
- 멱등성, 보안 오버레이, 네트워크 격리 같은 내부 구조

## 이 책이 다루지 않는 것

- Windows / Linux 환경 (지원 예정 자료는 본문 내 각주로만 안내)
- 클라우드 LLM API 연동 자세한 비용 비교
- 고급 도커 네트워킹 이론
